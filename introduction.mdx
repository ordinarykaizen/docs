## Smart Routing

`POST /v1/smart/chat/completions`

```javascript
fetch('https://api.promptlab.ai/v1/smart/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'gpt-4',
    messages: [{ role: 'user', content: 'Hello!' }],
    client: {
      user: 'user-123' // Optional: Track individual end users
    }
  })
})
```
