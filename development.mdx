<Info>
  **Prerequisite**: Node.js (version 19 or higher)
</Info>

## Local Setup

**1. Install Dependencies**
```bash
npm install
```

**2. Start Development Server**
```bash
npm run dev
```
Preview at `http://localhost:3000`

## Custom Port
Add to package.json:
```json
"dev": "next dev -p 3333"
```

## Update Dependencies
```bash
npm update
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Build Errors">
    1. Clear cache: `npm run clean`
    2. Remove node_modules: `rm -rf node_modules`
    3. Reinstall: `npm install`
  </Accordion>
  <Accordion title="Port Already in Use">
    Kill the process and restart: `npm run dev`
  </Accordion>
</AccordionGroup>
