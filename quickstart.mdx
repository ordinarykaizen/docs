## Setup

### Development Environment
<AccordionGroup>
  <Accordion icon="github" title="Clone Repository">
    1. Find your docs repo on [dashboard](https://dashboard.promptlab.ai)
    2. Clone locally: `git clone <repository-url>`
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Local Preview">
    1. Install dependencies: `npm install`
    2. Run: `npm run dev`
  </Accordion>
</AccordionGroup>

### Deployment
<AccordionGroup>
  <Accordion icon="message-bot" title="GitHub App">
    1. Install from [dashboard](https://dashboard.promptlab.ai)
    2. Verify check mark appears next to commit hash
  </Accordion>
  <Accordion icon="rocket" title="Deploy Changes">
    1. Commit and push changes
    2. Auto-deployment via GitHub app
    3. Manual deploy available on dashboard
  </Accordion>
</AccordionGroup>
